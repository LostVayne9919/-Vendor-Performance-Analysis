# 📊 Vendor Performance Analysis

## 📌 Overview
This project focuses on analyzing vendor performance using sales and revenue data. The workflow includes data ingestion, data cleaning, exploratory data analysis (EDA), automated summary generation, and interactive dashboard creation using Power BI. The project concludes with a professional report containing actionable business insights and performance recommendations.

---

## 🚀 Features

- Automated ingestion and preprocessing of vendor sales data
- Data cleaning and transformation using Python
- Exploratory Data Analysis (EDA) to identify trends and performance patterns
- Automated vendor-wise summary report generation
- Interactive Power BI dashboard for performance monitoring
- Comprehensive PDF report with actionable insights
